# 🌿 Afiya — Connected Wellness AI

> *Your health data, finally explained. Your habits, gently guided.*

Afiya is a daily wellness coach for women. It connects to your **Oura Ring**, reads your sleep, recovery and activity data, and uses **Claude AI** to deliver one clear, personalized recommendation every morning — based on your real data, not generic advice.

No medical jargon. No overwhelming dashboards. Just a human-feeling insight, every day.

## Features

**Accueil — Daily Advice**
- Personalized AI recommendation generated by Claude every morning
- Based on your real Oura data: sleep duration, HRV, readiness score, steps
- Adapts to your cycle phase, hormonal profile and personal goals
- Ask Afiya any question and get a contextual answer

**Sommeil — Sleep Dashboard**
- 7-day sleep history with visual bar chart
- Readiness ring (score /100 with color coding)
- Deep sleep, REM, HRV, resting heart rate, sleep latency
- Readiness contributor breakdown

**Cycle — Cycle Tracker**
- Current phase display (menstrual / follicular / ovulation / luteal)
- 14-day calendar strip with phase colors
- Daily mood check-in and symptom log
- Next period prediction

**Profil — Settings**
- Oura token entry with live validation
- Anthropic API key for AI recommendations
- Hormonal profile (standard, SOPK, perimenopause, etc.)
- Personal health goals
- Full local data deletion (RGPD)

## Privacy

All data stays on your device. Afiya does not collect, store or share anything. Your Oura token and Anthropic key are stored in your phone's secure enclave. You can delete all data at any time from the Profile screen.

## Getting Started

**Requirements:** Node.js 18+, Expo Go app on your phone, Oura account, Anthropic API key

```bash
git clone https://github.com/your-username/Afiya
cd Afiya/mobile
npm install
npx expo start
```

Scan the QR code with Expo Go, then go to **Profil** to enter your Oura token and Anthropic key.
