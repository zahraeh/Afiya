# Connected Habit & Wellness AI

This project builds an AI-driven daily wellness coach that generates habit recommendations 
from health and lifestyle data (sleep, activity, cycle tracking, mood, etc.).

## Project Structure
- `data/`: raw and cleaned datasets
- `notebooks/`: analysis and modeling notebooks
- `src/`: scripts for cleaning and habit recommendation
- `app/`: streamlit dashboard

## Goals
- Collect wearable/app data (Oura, Flo, Fitbit, Apple Health)
- Explore correlations between lifestyle metrics and wellbeing
- Build rule-based and AI-driven recommendation system
- Deliver results through an interactive dashboard
